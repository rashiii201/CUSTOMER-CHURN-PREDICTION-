# CUSTOMER-CHURN-PREDICTION-
Customer Churn Prediction is a machine learning project that analyzes customer data to identify factors influencing churn. It includes data preprocessing, EDA, feature engineering, and classification models like Logistic Regression to predict churn and support data-driven retention strategies.

# FEATURES 
In this project, a complete machine learning workflow is implemented.

-Including data preprocessing, handling missing values

-Encoding categorical variables

-Feature scaling

-Model building

-And performance evaluation. (A Logistic Regression classification model is used along with a pipeline to automate preprocessing and training steps efficiently.)
